# Voqsy - социальное приложение для выкладки блогов

#### Функционал:
 - Авторизация и регистрация
 - Добавление, удаление, редактирование и чтение всех блогов (CRUD)
 - Комментарий и лайк (для авторизованных)
 - Создание новых блогов (для авторизованных)
 - Редактирование зарегистрированного аккаунта

***В данный момент приложение не готово к продакшену и находится в стадии разработки***
